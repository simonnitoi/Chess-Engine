# Simple Chess Engine
A simple brute force meterialistic chess algorithm.

## How it Works
- Uses the python chess library for all board logic and chess function.
- Sends board state to `GreedFish.py` which looks only at material, draws, and checkmates at a single ply.

## Setup
1. Clone the repository.
2. Install dependencies listed in requirements.txt.
3. Run python `main.py`.
